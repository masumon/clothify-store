Phase 1: Security Hardening
1. Admin mutation CSRF/Origin protection
- নতুন shared guard file: lib/admin-mutation-guard.ts
- proxy.ts এ /api/admin/* mutation request-এর জন্য trusted Origin/Referer validation যোগ করা হয়েছে
- effect: cross-site forged admin mutation request এখন explicit validation ছাড়া pass করবে না

2. Admin auth hardening
- lib/admin-auth.ts এ ADMIN_SESSION_SECRET fallback সরানো হয়েছে
- production-এ legacy HTTP Basic auth fallback disable করা হয়েছে
- next.config.js startup validation এ ADMIN_SESSION_SECRET, ADMIN_USERNAME, ADMIN_PASSWORD required করা হয়েছে
- lib/api/admin-health.ts এ missing secret এখন warning নয়, error হিসেবে report হয়

3. Browser security headers
- next.config.js এ CSP, X-Content-Type-Options, Referrer-Policy, Permissions-Policy, X-Frame-Options যোগ করা হয়েছে
- app/layout.tsx থেকে external Font Awesome CDN link সরানো হয়েছে
- inline preference bootstrap script external file public/preference-bootstrap.js এ নেওয়া হয়েছে

4. Invoice sanitization
- components/InvoiceAutoDownload.tsx raw HTML string interpolation এবং document.write() flow থেকে safe React-rendered popup invoice এ refactor করা হয়েছে
- popup head/body reset এখন replaceChildren() দিয়ে হচ্ছে

5. External icon dependency removal
- নতুন local icon wrapper: components/AppIcon.tsx
- lucide-react ভিত্তিক icon system বসানো হয়েছে
- Font Awesome usage codebase থেকে remove করা হয়েছে
